# 📈 211_rs — RS（相対力）スキャナー

日経225構成銘柄の相対力（RS）を日経225・TOPIXと比較し、
RS水準・RS転換・隠れ強気・セクター出遅れの4パターンで候補銘柄を Discord に通知します。

## 検出パターン

| パターン | 条件 |
|---|---|
| `[A]` RS水準 | いずれかの期間で RS > 1.15 |
| `[B]` RS転換 | 直近3日で RS が底打ち反転 |
| `[C]` 隠れ強気 | 株価5日リターン < 0 かつ RS20日 > 1.10 |
| `[S]` セクター出遅れ | セクター平均が上昇中に個別銘柄が出遅れ＋出来高増加 |

## 季節性フィルター（バックテスト 2015–2025）

除外月（平均リターンがマイナス）: **3月・6月・7月**
除外月はスキャンをスキップし Discord に通知します。
